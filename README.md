# ClassiCube Textures, Font, and Sounds for Minecraft
The ClassiCube look and feel, in Minecraft.

Includes [zoey (goodly)](https://github.com/Goodlyay)'s original textures and font, and [csnxs](https://github.com/DrinkyBird)'s soundpack (optional).

**Supports both Minecraft Java and Bedrock.**

# FAQ
* The crosshair is slightly larger than it is in ClassiCube.
	* This is due to rendering differences between ClassiCube and Minecraft. The crosshair in this pack is actually scaled 2x, because otherwise it'd actually end up being too small, and I figured having it be slightly bigger is the better choice of the two.
* Gold and iron blocks in Bedrock use the same textures for all sides.
	* This is a Bedrock limitation, since unlike Java, you apparently can't define custom models for vanilla blocks. So on Bedrock, all sides of the gold and iron blocks use the "top" texture.
