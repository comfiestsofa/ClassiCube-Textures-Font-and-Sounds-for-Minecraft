# ClassiCube Sounds for Minecraft
The ClassiCube soundpack, in Minecraft.

Includes just [csnxs (DrinkyBird)](https://github.com/DrinkyBird)'s soundpack.

**Supports both Minecraft Java and Bedrock.**

Looking for the full project with block and environment textures, and an optional soundpack? https://modrinth.com/project/k9HX9DGX

# FAQ
* **Minecraft Java says the resource pack is made for an older version of Minecraft!**
	* Ignore it, it'll work fine. This is because I set the lowest possible `pack_version` for each pack. The font uses a `pack_version` of 1, while everything else uses 4.
